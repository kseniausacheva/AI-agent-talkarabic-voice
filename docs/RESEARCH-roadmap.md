# Исследование: что добавить, граница с CRM, встраиваемость

Дата: 2026-06-10. Три направления: voice-AI инструменты продаж, CRM языковых
школ, техника встраивания в CRM. 36 фактов с источниками.

---

## Главный вывод

Идея инструмента **валидирована рынком**: «голос → структурированные поля CRM»
оформилось в отдельную категорию voice-to-CRM (Basata, CustomerIQ, snapAddy,
Leadbeam; Microsoft добавил Voice-to-CRM в Dynamics 365). Речь в ~3 раза
быстрее набора текста.

При этом ниша свободна: **ни одна из проверенных CRM для языковых школ
(AlfaCRM, Hollihop, Мой Класс, Параплан) не имеет ни голосового ввода, ни
LLM-чеклистов**. Ближайший конкурент — Bitrix24 BitrixGPT (транскрибация
звонка + автозаполнение карточки), но он работает только со звонками через их
телефонию (Telegram/WhatsApp-разговоры не покрыты) и стоит 9900 ₽/мес за 1000
AI-запросов. HubSpot с AI — от $450/мес. Наша механика «голосовой дебриф
менеджера ПОСЛЕ разговора по любому каналу» уникальна.

## Что добавить — три корзины по стоимости

### Корзина А: почти бесплатно (расширение JSON-ответа MiniMax, +1 промпт)

| Фича | Кто так делает | Что нужно |
|---|---|---|
| **lead_score 1–10 с обоснованием** | Gong, HubSpot | Портрет идеального ученика школы (4–5 предложений) в системный промпт. Точность AI-скоринга 40–60% против 15–25% у ручного |
| **objections[] — возражения** | Outreach, trata.ai | Фиксированный словарь: цена / время / техника / доверие / другое. Дашборд покажет топ возражений по школе |
| **follow_up_draft** | HubSpot Breeze, Kommo AI | Готовый черновик сообщения клиенту в WhatsApp (рус/араб) — менеджер копирует и шлёт |
| **tasks[] — задачи из разговора** | Fireflies | «Отправить расписание», «позвать на пробный в четверг» — в отдельную таблицу с дедлайном |
| **stage — стадия воронки** | Kommo AI | новый / тёплый / готов к оплате / отказ — по правилам школы |
| **next_contact_date** | стандарт рынка | LLM предлагает дату следующего касания; 80% продаж закрывается после 5+ касаний |

### Корзина B: вообще без LLM (чистый SQL по нашей SQLite)

- **«Сегодня связаться»** — блок на дашборде: клиенты с next_contact_date
  сегодня/просрочено. Вероятно, самая денежная фича для школы
- **«Клиент замолчал»** — аналог риск-индикатора Gong: тёплые лиды без
  обновлений N дней
- **Coaching по менеджерам** — какие из 10 вопросов менеджер чаще всего
  не выясняет (доля пропусков по полям). Аналог «missed discovery questions»
- **Score чеклиста (0–10)** — доля заполненных полей, в Sheets и на дашборд.
  Сравнение между менеджерами = QA-инструмент владельца

### Корзина C: UX-паттерны лидеров

- **Diff вместо перезаписи** (Bitrix24): повторная надиктовка по тому же
  клиенту показывает старое/новое значение с подтверждением
- **Подсказка дозаписать** (Gong): поле пустое → UI сразу предлагает записать
  15-секундное голосовое только по этому вопросу
- **Чеклист на экране записи** (CustomerIQ): результат появляется сразу, без
  перехода на другую страницу

### Технический риск из исследования

Whisper на **диалектном арабском** даёт WER 25–50% (на египетском). Если
менеджеры вставляют арабские фразы: (1) промпт-подсказка Whisper со словарём
школы, (2) MiniMax как пост-корректор, (3) дообученная модель
whisper-medium-egy с HuggingFace (drop-in замена).

## Граница с CRM: что НЕ делать

Территория CRM (плотно занята, от 590 ₽/мес — Мой Класс, AlfaCRM 1500 ₽/мес):

- ❌ Воронка сделок и канбан
- ❌ Расписание занятий, журнал посещений
- ❌ Абонементы, платежи, онлайн-касса
- ❌ Рассылки (Email/SMS/WhatsApp)
- ❌ Личные кабинеты учеников
- ❌ Зарплаты преподавателей

Рыночный паттерн: **conversation intelligence — это слой ПОВЕРХ CRM**
(как Noota поверх Pipedrive), а не сама CRM. Наша стратегия: структурировать
разговор и **экспортировать** (Sheets уже есть, дальше — заметки в карточку
CRM), а не строить воронку у себя.

## Встраивание в будущую CRM: да, и почти готово

Наша архитектура (headless FastAPI + JWT + Next.js) — ровно тот паттерн,
который используют встраиваемые инструменты 2026 (Embeddable, Grafana).

### Сценарии

| CRM | Как встраивается | Сложность |
|---|---|---|
| **Самописная CRM школы** | iframe `/embed/checklist?token=...`; CRM-бэкенд получает короткоживущий JWT через server-to-server endpoint | Самый простой — мы контролируем обе стороны |
| **Kommo** | Виджет (manifest.json + ~100 строк JS), наш iframe в правой панели карточки лида. Приватная интеграция без модерации, long-lived token до 5 лет | Дни работы |
| **Bitrix24** | Локальное приложение, placement.bind на вкладку карточки сделки; Bitrix сам открывает наш URL в iframe с ID сделки | Дни работы; нужен публичный URL (DO) |

### Что заложить СЕЙЧАС, чтобы потом не переписывать

1. **Маршрут `/embed`** — версия сессии без шапки/навигации, контекст из
   URL-токена, postMessage с проверкой origin
2. **Embed-токены** — endpoint выдачи короткоживущих JWT по API-ключу
   (server-to-server). Внимание: third-party cookies в iframe мертвы
   (Chrome блокирует) — только Bearer/URL-токены. У нас уже Bearer ✓
3. **CSP frame-ancestors** — настраиваемый allowlist доменов через env
   `EMBED_ALLOWED_ORIGINS`
4. **Поля `external_source` / `external_entity_id`** в таблице checklists +
   фильтр в API — связь нашего чеклиста со сделкой в любой CRM
5. **Таблица `integration_credentials`** — OAuth-токены CRM + экспортные
   адаптеры (заметка в карточку Kommo/Bitrix наравне с Google Sheets)

Маркетплейсы (если инструмент станет продуктом для других школ): Kommo —
бесплатно с модерацией; Bitrix24 Market — через партнёрку, revenue-share до 50%.

## Рекомендация приоритетов

1. **Сейчас (1 день):** корзина A — расширить JSON-схему MiniMax
   (score, возражения, follow-up, задачи, дата касания). Один промпт,
   максимум ценности
2. **Следом (1 день):** корзина B — «Сегодня связаться» + score на дашборд
3. **При деплое на DO:** заложить 5 пунктов встраиваемости (§выше) — копейки
   сейчас, недели экономии потом
4. **CRM не строить** — школе дешевле Мой Класс/AlfaCRM за 600–1500 ₽/мес +
   наш инструмент как intelligence-слой. Самописная CRM имеет смысл только
   если станет продуктом — и тогда наш инструмент в неё вшивается за дни
